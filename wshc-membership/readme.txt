Developed by the World Sports Health Council (WSHC), the leading global authority in Sports Health and Medical Excellence. This plugin serves as the digital infrastructure for academic verification, professional certification, and scientific research management, bridging the gap between global health standards and digital innovation.

== Description ==

The WSHC Membership system is an elite digital portal designed for healthcare professionals, researchers, and academic institutions within the sports health sector.

Key Features:
* Unified AJAX-powered dashboard architecture.
* 11-tier professional role hierarchy.
* Secure multi-step registration with OTP verification.
* Public-facing professional "Scientific CV" profiles.
* Administrative "User Control Center" for institutional management.
* Monochromatic high-end monochromatic UI.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wshc-membership` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. The plugin will automatically create the required /login and /account pages.
